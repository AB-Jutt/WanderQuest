# WanderQuest 🌍

WanderQuest is a responsive trip planner web application built using HTML, CSS, and JavaScript. The application helps users discover travel destinations based on their current location, view weather information, save favorite spots, and explore destinations through a responsive image gallery. It provides a simple and interactive way to plan and organize trips while demonstrating modern front-end web development concepts.

## Features

- 📍 Location-based destination search using the Geolocation API
- 🌤️ Real-time weather information
- ❤️ Save and manage favorite destinations
- 🖼️ Responsive image gallery
- 📱 Mobile-friendly and responsive design
- 📝 Interactive forms and user-friendly interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Geolocation API

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Launch `index.html` in any modern web browser.
4. Allow location access when prompted to use location-based features.
